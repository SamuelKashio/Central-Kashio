## 🚀 Qué Hacer AHORA (3 Pasos)

### 1️⃣ Crea un Nuevo Repositorio en GitHub Corporativo

```
GitHub → + (esquina superior izq) → New repository
Nombre: cdr-dashboard
Visibility: Private
NO inicialices con README
Click: Create repository
```

### 2️⃣ Sube los 5 Archivos

En tu nuevo repo vacío:

```
Click: "Add file" → "Upload files"
Sube: app.py, requirements.txt, .gitignore, README.md
Para .streamlit/config.toml:
  - Primero crea carpeta .streamlit
  - Luego sube config.toml dentro
Commit message: "Initial commit: Dashboard CallMyWay v7.0"
Click: Commit changes
```

### 3️⃣ Configura Secretos en Streamlit Cloud

```
https://share.streamlit.io
Abre tu dashboard → ⋮ → Manage app
Settings → Secrets
Pega:
  CMW_USER = "tu_usuario_callmyway"
  CMW_PASS = "tu_contraseña_callmyway"
Click: Save
```
